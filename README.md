# react-webpack-redux
react + webpack + redux minimal boilerplate



### Installing

install npm modules

```
npm install
```


### Running dev server

```
npm start
```

### Install Redux Devtools for chrome
https://chrome.google.com/webstore/detail/redux-devtools

happy coding!
